# Google Drive — IronHub Master

The `IronHub Master` folder on Google Drive is the central document repository for IronHub. It contains all key business documents organized by category.

## Folder Structure

| Folder | Contents |
|---|---|
| `Agreement Templates` | Affiliate agreements, commission agreement templates (Feb & Dec 2025 versions) |
| `Assessments` | Asset evaluations (e.g., TransAlta Kaybob 3, Keyera Bigoray) |
| `Business Plan` | IronHub Business Overview (Sept 2017) |
| `Business Resources` | General business reference materials |
| `Clients` | Individual folders for 60+ client accounts (see below) |
| `Development` | Technical development documents |
| `Finance / Accounting` | Financial records |
| `Human Resources` | Employment agreements (e.g., Mathieu Spehler) |
| `Images / Logo` | Brand assets (PNG, JPG, EPS, AI formats) |
| `Legal` | Confidentiality agreements and legal documents |
| `Management Update` | Historical bi-weekly and weekly meeting notes (2017–2018) |
| `Marketing` | Marketing materials and campaigns |
| `Operations` | Operational documents |
| `RFP Submission Documents` | Quality Assurance Plan, Subcontracting Plan, Sustainability Strategy, Safety Policy, OHS Program |
| `Sales Material` | Decks, flat sheets, pricing info, demo files |
