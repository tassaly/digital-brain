# Projects

Short-horizon work with a clear **outcome** and **deadline**.

A project is something you're actively working on right now that will be done at a specific point.

## Examples
- Launch v2 of product X by June
- Write and publish article on topic Y
- Complete online course Z

## File convention
Each project gets its own note: `YYYY-MM-DD_project-name.md`
