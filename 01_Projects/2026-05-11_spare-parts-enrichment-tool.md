# Spare Parts Enrichment Tool

This is the primary active project for IronHub. It involves developing an AI-powered tool to ingest, parse, and enrich large datasets of spare parts and surplus equipment.

## Project Overview
- **Goal:** Build a robust AI tool that can take raw data (e.g., bulk spreadsheet imports), parse it, and enrich it with standardized inventory options, providing a confidence level for the enriched data.
- **Current Status:** In active development (as of Q2 2026, roughly halfway through a 20-week development timeline).
- **Key Features:**
  - Bulk import capability with AI parsing.
  - Confidence level scoring for enriched data.
  - User review interface for low-confidence results, allowing manual edits or re-enrichment with additional text instructions.
- **Future Phases:**
  - Integrate OEM data via APIs to significantly increase data confidence.
  - Implement Optical Character Recognition (OCR) to process images of nameplates.
  - Map the enriched data seamlessly to the existing IronHub database architecture.

## Strategic Alignment
This project directly supports the core mission of transitioning IronHub into an intelligent procurement platform. It is a critical component of the broader AI tool adoption strategy outlined in [[02_Areas/ironhub-company]].
